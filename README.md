# 🎯 Project Objectives

✔ Collect real-time cricket match data

✔ Store live data in a relational database

✔ Perform SQL-based analytics

✔ Visualize match statistics interactively

✔ Build a complete end-to-end analytics dashboard

---

# ✨ Key Features

## 🔴 Live Match Dashboard

* Live and recent match updates
* Team names, scores, and match status
* Automatic data retrieval using the Cricbuzz API
* Automatic database updates

---

## 📊 Analytics Dashboard

* Interactive KPI cards
* Match statistics
* Team distribution charts
* Recent match summaries
* Interactive visualizations

---

## 🗄️ SQL Analytics

* Predefined SQL query library
* One-click query execution
* Tabular results
* Automatic chart generation
* Database exploration

---

## 🛠️ CRUD Operations

* Add new match records
* View existing records
* Update stored information
* Delete unwanted entries

---

# 🔄 Application Workflow

```text
Cricbuzz API
      │
      ▼
Fetch Live Match Data
      │
      ▼
Store in SQLite Database
      │
      ▼
Execute SQL Queries
      │
      ▼
Generate Analytics
      │
      ▼
Interactive Streamlit Dashboard
```

```bash
pip install -r requirements.txt

streamlit run app.py
```

> **Note:** Configure your Cricbuzz (RapidAPI) API key before launching the application.
